Welcome to the Automata Simulator

An automata is defined by its five-tuple, the notation is A = (Q, ∑, δ, q₀, F)

Where

  • Q = A finite set of states.

  • ∑ = A finite set of input symbols.

  • δ = A transition function that takes as arguments a state and an input symbol and returns a state.

  • q₀ = The initial state.

  • F = A set of final or accepting states. (F is a subset of Q)
